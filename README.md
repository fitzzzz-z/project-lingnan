# Project Lingnan

**AI × Cross-border Wealth Management**

---

## 1. Vision（愿景）

成为具备：

- 金融市场理解能力
- 跨境财富管理能力
- AI Agent 构建能力

的复合型专业人才。

长期目标：

面向粤港澳家庭、企业和财富管理机构，
提供智能化、系统化的财富管理解决方案。

---

## 2. Background（背景）

当前基础：

- 两年股票投资经验
- 对金融市场形成初步理解
- 正在学习 AI Agent 和自动化工作流

未来方向：

AI + 金融 + 横琴跨境金融。
