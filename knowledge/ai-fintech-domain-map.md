# AI 金融科技总领域地图 V1.0

本地图是 Project Lingnan 的总知识入口。它服务于职业路线“AI 金融科技 → 横琴跨境金融生态 → 财富管理解决方案”，不代替各子领域的详细地图。

## 1. 金融机构与业务

### 机构

- 银行
- 证券与基金
- 保险
- 财富管理机构
- 金融科技公司
- 监管与公共服务机构

### 业务流程

- 客户准入与尽职调查
- 产品研究与适当性
- 投资与资产配置
- 风险管理与合规
- 客户服务与运营
- 报告、审计与知识管理

## 2. AI、数据与工程

### 基础能力

- Python
- API
- Excel 与 pandas
- 数据清洗和基础分析

### AI 应用

- LLM 与 Prompt Engineering
- RAG 与知识库
- Agent 与 MCP
- 工作流自动化
- 评估、监控与人工复核

### 数据治理

- 数据来源与授权
- 隐私与敏感信息
- 数据质量
- 版本与更新时间
- 审计记录

## 3. 横琴与跨境金融

### 法域与机构

- 内地及珠海
- 横琴粤澳深度合作区
- 澳门
- 香港

### 核心议题

- 横琴政策体系
- 跨境资金流动
- 外汇与账户体系
- 跨境支付与结算
- 跨境投资和财富服务
- 数据跨境与监管协同

## 4. 产品、用户与合规

### 产品能力

- 用户与利益相关者识别
- 业务痛点和流程分析
- 需求定义
- 原型和项目管理
- 用户验证与迭代

### 风险边界

- 金融业务资质
- 投资者适当性
- 法律与税务专业边界
- 模型错误与可解释性
- 隐私和信息安全
- 人工复核与责任归属

## 5. 财富管理应用

- 客户信息和需求整理
- 家庭目标、现金流与期限分析
- 资产配置研究辅助
- 风险识别与情景测试
- 政策与产品知识检索
- 顾问工作流和客户服务支持

详细内容见 [跨境财富管理知识地图](wealth-management-map.md)。

## 6. 证据与来源标准

### 来源优先级

1. 法律法规、监管机构和政府官方文件
2. 金融机构正式披露、行业标准和学术研究
3. 可信专业机构的研究报告
4. 新闻、评论和二手材料

### 每条重要结论应记录

- 来源链接或文件
- 发布机构
- 发布或生效日期
- 适用法域和对象
- 事实、推断或个人判断
- 可能失效或需要更新的条件

### 项目证据

- 可运行或可演示的成果
- 准确性和失败案例
- 业务用户反馈
- 合规与隐私边界
- 相比原流程的质量、风险或效率变化
